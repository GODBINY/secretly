# Secretly Chat 💬

회사 내부용 간단한 실시간 채팅 애플리케이션

## 특징

- 🏠 **다중 채팅방**: 여러 개의 채팅방 생성 및 참여 가능
- ⚡ **실시간 통신**: Socket.io 기반 실시간 메시지 전송
- ⌨️ **타이핑 인디케이터**: 상대방이 입력 중인지 실시간으로 확인
- 📌 **공지 및 답변**: 각 방마다 공지를 올리고 답변을 받을 수 있음
- 🔔 **귀여운 알림**: 작고 귀여운 시스템 알림
- 🖥️ **로컬 서버**: 개발자의 로컬 머신이 서버 역할

## 설치 및 실행

### 1. 의존성 설치

```bash
npm install
```

### 2. 서버 실행

```bash
npm run server
```

서버가 실행되면 다음 주소로 접속 가능합니다:
- 로컬: `https://localhost:3000`
- 네트워크: `https://[서버IP]:3000`

**참고**: 자체 서명 인증서를 사용하므로 브라우저에서 보안 경고가 표시될 수 있습니다. "고급" → "계속 진행"을 선택하시면 됩니다.

### 3. 클라이언트 실행

새 터미널에서:

```bash
npm start
```

또는 서버와 클라이언트를 동시에 실행:

```bash
npm run dev
```

## 사용 방법

1. 앱 실행 시 닉네임을 입력합니다.
2. 사이드바에서 방을 선택하거나 새 방을 만듭니다.
3. 메시지를 입력하여 채팅합니다.
4. 상단의 "공지 등록" 버튼으로 공지를 올릴 수 있습니다.
5. 공지가 있으면 답변을 작성할 수 있습니다.

## 네트워크 설정

팀원들이 접속하려면:

1. 서버를 실행한 컴퓨터의 로컬 IP 주소를 확인합니다.
2. 방화벽에서 포트 3000을 열어둡니다.
3. 팀원들에게 서버 주소를 구두로 공유합니다 (예: `https://192.168.1.100:3000`).
4. 팀원들이 앱을 실행하면 처음에 서버 주소를 입력하는 화면이 나타납니다.
5. 서버 주소를 입력하면 자동으로 저장되어 다음번 실행 시에는 자동으로 연결됩니다.
6. **중요**: 자체 서명 인증서를 사용하므로 Electron 앱에서 SSL 인증서 검증을 비활성화해야 합니다 (이미 설정됨).

## 기술 스택

- **Electron**: 데스크톱 애플리케이션
- **Node.js**: 서버 런타임
- **Express**: HTTP 서버
- **Socket.io**: 실시간 양방향 통신
- **HTML/CSS/JavaScript**: 프론트엔드
